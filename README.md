# Your GitHub Portfolio Setup Guide

This guide will help you create a professional GitHub portfolio to showcase your web development skills and projects. Follow these steps to get started:

### ✅ 1. Create a GitHub Account

- Go to [https://github.com](https://github.com)
- Sign up with your email and choose a professional username (e.g., `rahulborikar`)

---

### ✅ 2. Create a Special Portfolio Repository

- This is your personal GitHub profile page.
- Create a new repository named **exactly** like your username (e.g., `rahulborikar`)
- Keep it **public**
- Check the box “Add a README file”
- ✨ This README will be shown on your GitHub profile page. You can add your intro, skills, project links, and contact info here.

---

### ✅ 3. Update Your Profile README

Edit the README with this template:

```markdown
# Hi, I'm Rahul Borikar 👋

🎓 MCA Graduate | 💻 Full Stack Developer in Progress  
🔍 Actively learning HTML, CSS, JavaScript, React, Node.js & Python  
🛠️ Building real-world projects to showcase my skills

## 🧠 Skills

- HTML5, CSS3
- JavaScript (ES6+)
- Git & GitHub
- React (coming soon)
- Python (learning side-by-side)

## 📂 Projects

- [Portfolio Website](https://github.com/rahulborikar/portfolio)
- [To-Do App (JS)](https://github.com/rahulborikar/todo-app)
- [Weather App (API)](https://github.com/rahulborikar/weather-app)

## 📫 Connect with Me

- 📧 Email: rahul.borikar@example.com _(replace this with your real email)_
- 💼 LinkedIn: [Your LinkedIn URL here]
```

---

### ✅ 4. Start Adding Projects

For each project:

- Create a new repo (e.g., `portfolio-website`, `todo-app`)
- Upload your code:
  - Use Git commands (`git init`, `git add`, `git commit`, `git push`)
  - **OR** use GitHub Desktop for easier upload
- Add a `README.md` in each project folder explaining:
  - What the project is
  - Features
  - Technologies used
  - How to run it

---

### ✅ 5. Optional: Deploy Your Portfolio Site

Once you finish your personal portfolio website (HTML/CSS/JS), deploy it on **GitHub Pages**:

- Push the code to a new repo (e.g., `portfolio`)
- Go to **Repo Settings > Pages**
- Set source to `main` branch + `/root`
- GitHub will give you a URL like:  
  `https://rahulborikar.github.io/portfolio/`

---

### ✅ 6. Pin Your Best Projects

- Go to your GitHub Profile → “Customize your pins”
- Pin your 6 best repos (portfolio, projects, etc.)
